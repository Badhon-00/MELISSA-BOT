## 🚀 Quick Setup

```bash
# Clone the repository
git clone https://github.com/Badhon-00/ayanesan-gift.git

# Navigate to directory
cd ayanesan-gift

# Install dependencies
npm install

# Configure your settings
cp config.example.js config.js

# Start your bot
npm start
```
